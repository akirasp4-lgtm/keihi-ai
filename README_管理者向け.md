# 経費精算アプリ 使い方ガイド（管理者向け）

---

## 管理者画面の開き方

PCのブラウザで以下のURLを開いてください。

**→ `https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/admin.html`**

> このURLは管理者のみに共有してください。社員には社員用URL（`/`）だけ共有してください。

---

## 経費データの取込

1. 社員からLINEやメールで送られてきたJSONファイルを保存する
2. 管理者画面の**ドロップゾーンにファイルをドラッグ＆ドロップ**（または クリックして選択）
3. 複数ファイルを一度に取り込めます

取込後は人物ごとにレシート画像・金額・項目が一覧表示されます。

---

## JDL形式CSVの出力

1. 画面右上「**JDL形式CSV出力**」ボタンをクリック
2. `JDL取込_YYYY-MM-DD.csv` がダウンロードされる
3. JDL会計ソフトにCSVを取り込む

### CSVの列構成（JDL仕様）

| 列 | 内容 |
|---|---|
| 識別フラグ | `1000` 固定 |
| 伝番 | 連番（001〜） |
| 日付 | YYYYMMDD形式 |
| 借方科目名称 | 選択項目に対応した科目名 |
| 借方補助名称 | 補助科目名（駐車場代・ガソリン等） |
| 借方課区 | 仕入 / 対象外 |
| 借方税区 | 10% / 軽減8% / 対象外 |
| 借方税入力方法 | 内税 固定 |
| 借方金額 | 税込金額 |
| 借方消費税 | 自動計算 |
| 貸方科目名称 | 現金 固定 |
| 摘要 | 店名 ＋ [元請け / 現場] ＋ メモ |
| 借方部門名称 | 申請者名 |

## GitHub Pagesで公開する手順（初回のみ）

1. [GitHub](https://github.com) で **New repository** を作成（Public）
2. `index.html`・`admin.html`・`README.md`・各READMEをアップロード
3. `Settings` → `Pages` → **Branch: main / (root)** → Save
4. 数分後に `https://ユーザー名.github.io/リポジトリ名/` でURLが発行される
5. 社員用URL（`/`）を社員全員に共有する
6. 管理者用URL（`/admin.html`）は管理者のみに共有する

---

## 注意事項

- 取込データはPCのブラウザ内に保存されます
- ブラウザのデータを削除するとデータが消えます。CSV出力は定期的に行ってください
- 同じJSONを重複して取り込んでも重複しないよう処理されています

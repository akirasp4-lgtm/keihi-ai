# 経費精算アプリ 使い方ガイド（社員向け）

---

## アプリの開き方

スマホのブラウザで以下のURLを開いてください。

**→ `https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPO_NAME/`**

ホーム画面に追加しておくと次回から楽です。

---

## 初回のみ：Gemini APIキーの設定

OCR（レシートの自動読み取り）を使うために、**無料のAPIキーを各自で取得**して設定します。

### キーの取得手順

1. [Google AI Studio](https://aistudio.google.com/app/apikey) をスマホで開く
2. Googleアカウントでログイン
3. 「Create API key」をタップ
4. 「Create API key in new project」を選択
5. 表示されたキー（`AIzaSy...`）をコピー

### キーの設定手順

1. アプリ右上の「Gemini APIキー」をタップ
2. コピーしたキーを貼り付けて「保存」

> **無料枠**：1日1,500回・毎分15回まで無料（経費精算には十分です）

---

## 経費の入力手順

### 1. レシートを撮影する

- 「カメラで撮影」または「写真を選択」をタップ
- 撮影後、日付・金額・店名が**自動で入力**されます

### 2. 内容を確認・修正する

| 項目 | 内容 |
|---|---|
| 日付 | 自動入力（必要に応じて修正） |
| 金額（税込） | 自動入力（必要に応じて修正） |
| 税率 | 飲食は「軽減8%」に変更 |
| 項目 | プルダウンから選択 |
| 店名・購入場所 | 自動入力（必要に応じて修正） |
| 元請け名 | 例：〇〇建設 |
| 現場名 | 例：〇〇邸 |
| メモ | 任意 |

### 3. 保存する

「保存して次へ」をタップするとリストに追加されます。

---

## 管理者への送付

1. 画面下「リスト」タブをタップ
2. 「JSON出力（管理者へ送付）」をタップ
3. ダウンロードされたファイルを管理者にLINEやメールで送付

---

## 注意事項

- データはこのスマホの中にのみ保存されます
- ブラウザのデータを削除するとデータが消えます。定期的にJSON出力して管理者に送ってください

経費精算アプリ セットアップ手順書

管理者向け・初回セットアップ完全ガイド

全体の仕組み

社員のスマホ 　↓ レシート撮影 GitHub Pages（index.html）※APIキーなし・安全 　↓ Cloudflare Worker（keihi-ocr）※APIキーをここに保管 　↓ Groq API（無料・クレカ不要） 　↓ OCR結果を返す 社員の画面に自動入力
✅ 社員はAPIキー設定不要
✅ APIキーはGitHubに入らないので安全
✅ Groqは無料・クレジットカード不要（1日14,400回）

STEP 1：GitHub Pagesで公開

1
github.com でアカウント作成（無料）
2
新しいリポジトリを作成（Public・名前は keihi-ai など）
3
全HTMLファイルをリポジトリにアップロード
4
Settings → Pages → Branch: main / (root) → Save
5
数分後にURLが発行される

アプリURL

https://akirasp4-lgtm.github.io/keihi-ai/
https://akirasp4-lgtm.github.io/keihi-ai/admin.html

STEP 2：Groq APIキーの取得（無料・クレカ不要）

1
console.groq.com を開いてGoogleアカウントでサインアップ
2
左メニュー「API Keys」→「Create API key」をクリック
3
Display Name に keihi と入力して「Submit」
4
gsk_ で始まるAPIキーをコピーして保管
⚠️ APIキーは作成時しか表示されません。必ずコピーして保管してください。

STEP 3：Cloudflare Workerの作成

1
cloudflare.com で無料アカウント作成
2
Workers & Pages → Create → Start with Hello World!
3
Worker名を keihi-ocr にして「Deploy」
4
「Edit code」を押して下記コードを全て貼り付けて「Deploy」

Workerコード

export default { async fetch(request, env) { if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST, OPTIONS', 'Access-Control-Allow-Headers': 'Content-Type', } }); } if (request.method !== 'POST') { return new Response('Method Not Allowed', { status: 405 }); } const body = await request.json(); const b64 = body.contents?.[0]?.parts?.[0]?.inlineData?.data || ''; const mime = body.contents?.[0]?.parts?.[0]?.inlineData?.mimeType || 'image/jpeg'; const prompt = body.contents?.[0]?.parts?.[1]?.text || ''; const groqRes = await fetch('https://api.groq.com/openai/v1/chat/completions', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${env.GROQ_API_KEY}`, }, body: JSON.stringify({ model: 'meta-llama/llama-4-scout-17b-16e-instruct', messages: [{ role: 'user', content: [ { type: 'image_url', image_url: { url: `data:${mime};base64,${b64}` } }, { type: 'text', text: prompt } ] }], temperature: 0, }), }); const data = await groqRes.json(); if (!groqRes.ok) { return new Response(JSON.stringify({ error: { message: data?.error?.message || `HTTP ${groqRes.status}` } }), { status: groqRes.status, headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*' } }); } const text = data.choices?.[0]?.message?.content || ''; return new Response(JSON.stringify({ candidates: [{ content: { parts: [{ text }] } }] }), { headers: { 'Content-Type': 'application/json', 'Access-Control-Allow-Origin': '*' } }); } };
5
Settings → Variables and Secrets → 「+ Add」をクリック
6
Variable name: GROQ_API_KEY / Value: gsk_... のキー / Type: Secret → Deploy

APIキーを変更したいとき

1
console.groq.com で新しいAPIキーを作成
2
Cloudflare → keihi-ocr → Settings → Variables and Secrets
3
GROQ_API_KEY の編集ボタンを押して新しいキーに更新 → Deploy

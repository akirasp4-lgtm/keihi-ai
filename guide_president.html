経費入力のながれ

  1. 1「📸 カメラで撮影」または「🖼️ 写真を選択」でレシートを取込
  2. 2AIが解析 → 日付・金額・店名・税率が自動入力（数秒）
  3. 3項目（カテゴリ）を選ぶ
  4. 4💳 支払元を選ぶ（現金 / GR / GRHD）
  5. 5🏢 費用負担は初期「按分しない」。按分したい時だけ「按分する」に切替
  6. 6「保存して次へ」をタップ
⚡ 普通の経費は「按分しない」のままでOK。会社間で割り振る時だけ「按分する」を押してください。

💳 支払元の選び方

どのカード/現金で払ったかを選びます。必須です。

選択肢どんなとき
💴 現金現金で払ったとき（立替）
💳 GRグローライズのアメックスで払ったとき
💳 GRHDGRHDのアメックスで払ったとき
💡 会計ソフト側で自動的に「現金」か「未払金」科目に振り分けられます。

🏢 費用負担（按分する/しない）

2つのトグルから選ぶだけ。どちらかに切り替えます。

モード動作
按分しない（初期）会社情報なしの通常の経費として保存。事務側はそのまま使います
按分する6社に%で振り分ける入力UIが開く

按分する時の操作

  1. 1「按分する」をタップ
  2. 2負担する会社のチェックを入れる
  3. 3それぞれに %を入力（合計100%になるように）
  4. 4合計100% ✅ を確認して保存

便利ボタン

ボタン動作
⚖ 均等割チェックした会社で均等割り（3社なら33/33/34%）
🗑 クリア全部未選択にする

最近の按分パターン

よく使う按分は「最近」ボタンで再表示されます（直近3件まで自動保存）。
例: GR50%/GRHD50% をよく使うと次回からワンタップで呼び出し。

金額指定モード

%ではなく具体的な金額で按分したい場合は「金額指定」に切替。
例: 総額¥10,000 → GR¥3,000 / GRHD¥7,000（合計が総額と一致しないと保存不可）

入力の具体例

例1: 普通の経費（按分しない）

撮影 → 項目選択 → 💳支払元タップ → 按分しないまま保存
→ 事務側は会社情報なしの通常経費として処理

例2: ¥10,000の会食を GR/GRHD 半々で按分

撮影 → 交際費 → 💳GRアメックス → 「按分する」タップ
→ ☑GR 50%、☑GRHD 50% → 合計100% ✅ → 保存

例3: 3社で均等割

「按分する」 → ☑3社チェック → ⚖均等割 → 保存
（33/33/34% に自動配分）

管理者への送付

  1. 1画面下の「📋 リスト」タブをタップ
  2. 2保存されたデータ一覧を確認（支払元と按分も表示されます）
  3. 3「📤 管理者に送信」をタップ
  4. 4「✅ Googleドライブへの送信が完了しました！」と出たら完了
  5. 5「リストを削除しますか？」→「OK」でリストを空にする
💡 データはGoogleドライブの「経費申請 / 小林社長」フォルダに自動保存されます。

保存できない場合のチェック

以下を確認してください:

  • 日付が入っているか
  • 金額が1円以上になっているか
  • 店名が入っているか
  • 💳支払元を選んでいるか
  • 🏢費用負担（単独負担時は1社選択、按分時は合計100%/総額一致）

この按分データは経理でどう使われるか

  1. 1管理者が月次で「JDL形式CSV」を出力
  2. 2按分ありの経費は会社別に複数行に分解されて仕訳される
  3. 3管理者が「Amex照合CSV」を出力
  4. 4アメックス明細と突合 → 他社負担分を自動で集計し、立替金請求の仕訳を作成
🎯 これで月末の手打ち計算が不要になります
📱 データはこのスマホの中にのみ保存されます。
ブラウザのデータを消すとデータも消えるため、こまめに「管理者に送信」を押して送ってください。
